This project is a collection of six arcade mini-games built around a chicken theme. Everything was developed by our team, from the games themselves to the physical controller used to play them.
The games are made using **HTML, CSS and JavaScript**, while the controls come from a custom-built electronic controller that lets players interact with the games in a more fun and unique way.
Each mini-game has its own gameplay and local leaderboard, where high scores are saved using the browser's Local Storage, so no database or internet connection is needed.

Features
* 6 different arcade mini-games
* Custom electronic controller
* Built with HTML, CSS and JavaScript
* Local high score system
* No external database
* Fully developed by our team
  
This project was created to combine web development with electronics and to build a complete arcade experience that is both fun and interactive.
